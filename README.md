# Сервис для работы с платежами по заказу

## Описание

Это проект для работы с платежами по заказу.  

Система позволяет выполнять оплату заказов одним или несколькими платежами, 
учитывать различные типы платежей (наличные и эквайринг), а также выполнять возвраты.

Статус оплаты заказа автоматически обновляется в зависимости от
суммы платежей.

Поддерживается интеграция с внешним банковским API для работы с эквайрингом.
Сервис может:
- инициировать платёж в банке;
- хранить идентификатор банковского платежа;
- проверять состояние платежа на стороне банка;
- синхронизировать статус платежа между системой и банком.

Проект реализован с использованием многослойной архитектуры.

### Слои приложения

- **API слой** — REST-эндпоинты FastAPI, обработка HTTP-запросов и ответов.
- **Сервисный слой** — бизнес-логика работы с заказами и платежами.
- **Репозитории** — доступ к базе данных.
- **Интеграционный слой** — клиент для взаимодействия с внешним банковским API.
- **Модели** — ORM-модели SQLAlchemy.
- **Схемы** — Pydantic-схемы для валидации входных и выходных данных.

### Основные возможности

- создание заказов;
- выполнение платежей по заказу;
- поддержка нескольких платежей для одного заказа;
- возврат платежей;
- автоматическое обновление статуса оплаты заказа:
  - `UNPAID`
  - `PARTIALLY_PAID`
  - `PAID`;
- интеграция с банковским API для эквайринга;
- обработка ошибок и доменных исключений;
- управление транзакциями через паттерн **Unit of Work**.

### Технологии

- Python 3.13
- FastAPI
- Pydantic
- Alembic
- SQLAlchemy 2
- PostgreSQL
- Docker

## Активация и запуск проекта

Клонируйте репозиторий:  

  ```bash
    git clone https://github.com/SlavaKlkv/aitiguru
  ```
    
Перейдите в корень проекта:
  ```bash
    cd aitiguru
  ```

### Запуск через Docker Compose

Запуск приложения
    
```bash
  docker compose up
```

Для запуска без пересборки образа с пересозданием контейнеров даже,  
если не было изменений:
```bash
  docker compose up --build --force-recreate
```

Для пересборки образа без использования кеша:
```bash
  docker compose build --no-cache
  docker compose up
```

Остановка всех контейнеров:

- Если запущено обычно:
```bash
  Ctrl + C
```

- Если запущено в фоне:
```bash
  docker compose down
```

### Запуск локально

Установите uv:

  ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
  ```

Установите зависимости:

  ```bash
    uv sync
  ```

Запуск uvicorn с автоперезагрузкой после сохранения обновленного кода:

  ```bash
    uvicorn src.main:app --reload --port 8000
  ```

### Открыть Swagger UI:
   'http://127.0.0.1:8000/docs'
